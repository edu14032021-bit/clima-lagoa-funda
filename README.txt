# Clima Lagoa Funda - GitHub Pages

Arquivos prontos para publicar como PWA no GitHub Pages.

Esta versão inclui:
- Tela de abertura / splash screen
- index.html com manifest e service worker registrados
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png

Envie todos estes arquivos para a raiz do repositório:

- index.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png

Depois ative o GitHub Pages em Configurações > Páginas > Implantar de uma branch > main/principal > /root.

Importante: após enviar uma nova versão, talvez seja necessário atualizar a página ou limpar o cache do navegador, porque o service worker pode manter arquivos antigos por alguns minutos.
