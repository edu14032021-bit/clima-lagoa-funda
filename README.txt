# Clima Lagoa Funda - GitHub Pages

Arquivos prontos para publicar como PWA no GitHub Pages.

Envie todos estes arquivos para a raiz do repositório:

- index.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png

Depois ative o GitHub Pages em Settings > Pages > Deploy from a branch > main > /root.
